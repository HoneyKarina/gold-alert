## 5. 技术社区帖子（详细版）

### 标题
[开源] Gold Alert - 一个金价监控预警工具

### 正文

## 背景

最近黄金价格波动很大，想做一个自动监控的工具。研究了一圈，发现没有特别简单好用的开源方案，于是自己动手写了 **Gold Alert**。

## 功能

### 1. 实时监控
- 从 Kitco 获取国际金价
- 支持自定义检查间隔
- 自动记录历史数据

### 2. 智能预警
- 设置价格阈值（如：涨跌 $50）
- 设置百分比阈值（如：涨跌 1%）
- 支持双向预警（涨 + 跌）

### 3. 多渠道通知
- 飞书（Lark）私聊通知
- Webhook 自定义通知
- 控制台输出

### 4. API 接口
```bash
# 获取当前金价
GET /api/price

# 获取历史数据
GET /api/history?days=7

# 设置预警规则
POST /api/alerts
```

### 5. Web 界面
- 简洁的仪表盘
- 响应式设计
- 实时更新

## 技术栈

- Node.js 18+
- TypeScript
- Express
- Axios

## 快速开始

```bash
# 克隆
git clone https://github.com/HoneyKarina/gold-alert.git

# 安装
npm install

# 配置
cp .env.example .env
vim .env

# 运行
npm start
```
