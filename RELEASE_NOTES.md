## Gold Alert - Open Source Gold Price Monitoring Tool

### 🎯 What is Gold Alert?

A lightweight, open-source tool for monitoring gold prices and getting notified when prices change significantly.

### ✨ Features

- 🔄 **Real-time Monitoring** - Fetches gold prices from Kitco
- 🚨 **Smart Alerts** - Customizable threshold and percentage-based alerts
- 📢 **Multi-channel Notifications** - Feishu, Webhook, Console output
- 📊 **Historical Data** - Track price changes over time
- 🌐 **Web Dashboard** - Beautiful, responsive interface
- ⚡ **REST API** - Easy integration with other applications

### 🚀 Quick Start

```bash
# Clone
git clone https://github.com/HoneyKarina/gold-alert.git
cd gold-alert

# Install
npm install

# Configure
cp .env.example .env
# Edit .env with your settings

# Run
npm start
```
